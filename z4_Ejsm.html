1. Kiedy powstał JavaScript (podaj datę)

    2. JavaScript powstał w 1995 roku

  2. Dlaczego jest to język skryptowy

    3. JavaScript jest przede wszystkim językiem skryptowym – to znaczy interpretowanym. Nie musi zostać skompilowany do kodu maszynowego, aby można było zobaczyć efekty jego działania

  3. Gdzie jest wykonywany JS (klient lub serwer)

    4. JS jest wykonywany u klienta

  4. dlaczego jest JS to bezpieczne narzędzie

    5. JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera

  5. czy jest to język obiektowy , jeśli tak to dlaczego

    6. Javascript jest językiem programowania obiektowego ponieważ wykorzystuje definicje obiektów, metod, zdarzeń, które mogą być zastosowane do opisu właściwości obiektów

  6. wymień zalety JS:

    7. Zaleta 1 Ponieważ jest to obecnie najbardziej rozpowszechniony język skryptowy, w Internecie można znaleźć miliony praktycznych przykładów – witryn wykorzystujących różnorodne funkcje JavaScriptu

    Zaleta 2 JavaScript pomaga użytkownikom tworzyć nowoczesne aplikacje internetowe do bezpośredniej interakcji bez ponownego ładowania strony za każdym razem

    Zaleta 3 Z pomocą frameworków (rodzaj programu do tworzenia innych programów w JS) zostały stworzone znane na całym świecie aplikacje, np. Google używa Angular, a Facebook — React

  7. Framework to:

    8. Rodzaj programu do tworzenia innych programów w JS

  8. Do aplikacji mobilnych można użyć Framework

    9. Bardzo popularnym frameworkiem do tworzenia aplikacji jest React Native. Z jego pomocą możemy budować aplikacje mobilne na różne systemy operacyjne (cross-platform apps)

  9. Aplikacje responsywne to:

    10. Responsywność JavaScript udostępnia metody, których można użyć w aplikacji, aby uzyskać bieżący rozmiar ekranu i wykorzystać wynik do wykonania działań w oparciu o ten rozmiar

  10. Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
    11. <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

    Kod ten tworzy przycisk z napisem "Nowa Strona" i przy kliknięciu wywołuje funkcję "WinOpen(' ')" która prawdopodobnie otwiera nowe okno przeglądarki

  11. Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
    12. window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

    • toolbar=no pokazuje lub ukrywa standardowy pasek narzędzi
    • directories=no pokazuje lub ukrywa przyciski katalogów
    • menubar=no pokazuje lub ukrywa menu przeglądarki
    • height=280 ustawia wysokość okna
    • width=160 ustawia szerokośc okna
    • top=200 ustawia położenie okna względem góry ekranu
    • left=200 ustawia położenie okna względem lewej strony ekranu

  12. Opisz następującą instrukcję window.close()

    13. Zamyka ona aktualne okno przeglądarki
